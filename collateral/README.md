# Source Assets

This folder holds reference material and original files related to the site — logos, brand assets, images, documents, and anything else worth keeping alongside the project but not publishing directly.

**Nothing in here appears on the website. The site is built from the `src/` folder only.**

Good things to put here:
* Original or high-resolution versions of images used on the site
* Logo files in various formats
* Screenshots, mockups, or design references
* Documents or collateral you want to keep with the project

If you're adding a file that should appear on the site, put it in `src/assets/` instead.
